In this guide, we will use the [Helius SDK](/helius/getting-started) to access the Solana connection object.

:::note
You can find a runnable example of this script in `snippets/helius` on <a href="https://github.com/pubkeyapp/pubkey-notes/tree/main/snippets/helius" target="_blank" rel="noopener noreferrer">GitHub</a>.
:::

### Access the Solana connection object

```typescript
// You can access the Solana connection object
const hash = await client.connection.getGenesisHash()

// Will print EtWTRABZaYq6iMfeYKouRu166VU2xqa1wcaWoxPkrZBG on devnet
console.log('Genesis hash:', hash)

// Get the version of the node
const version = await client.connection.getVersion()
console.log('Version:', version)
```
